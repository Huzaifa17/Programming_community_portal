# Programming Community Portal

A web-based platform where programmers can share educational blogs and learn from each other.

## 📌 Overview

- Inspired by platforms like :contentReference[oaicite:0]{index=0}  
- Users can write blogs on programming topics  
- Moderators review posts before publication

## ✨ Features

- 📝 Users can create and submit blogs  
- 🛡️ Admin and moderators can approve or reject posts  
- 🏠 Approved blogs appear on the home page  
- 🔼🔽 Upvote and downvote system for ranking posts  
- 🏆 Most useful blogs are shown at the top  
- 🔔 Notification system for recent posts and activities  

## 🛠️ Tech Stack

- Backend: Flask  
- Database: MongoDB  
- Frontend: HTML, CSS, Bootstrap
